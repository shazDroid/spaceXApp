## Project Description

<strong> Please install the app to see UI & beautiful animations & great creativity of arranging UI components etc.

* MVVM
* Dagger Hilt
* RoomDB
* Offline Cache
* Beautiful UI with lottie animations
* Motion Layout
* Data Binding 
* Clean & well defined code
* Almost no logic in fragment, activity everything is managed in ``xml`` & ``view model``
* Well used kotlin e.g Extension function, Higher Order Function, Scope Functions 
* Well defined & single resposibility of each module


<strong> Recycler View adapter are set in BindingAdapters
<strong> Ui validations like showing loading, error, success are all done by binding feature in ``xml`` itself.
